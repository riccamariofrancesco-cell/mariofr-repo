# mariofr-repo
MarioFR Stream does not host, store, or distribute any media content on our servers. We operate solely as a search and indexing platform that aggregates publicly available information from various third-party services.

We function as a web scraping tool that helps users discover content across different platforms. The app indexes and organizes information that is already publicly accessible on the internet.

Users are responsible for ensuring they have the right to access content through their local networks and we are not responsible for how users utilize the discovered content.
