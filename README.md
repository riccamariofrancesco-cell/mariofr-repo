# mariofr-repo

How to install our Add-on? The installation process is straightforward. Just download our latest repository zip file on zips > repository.mariofr > repository.mariofr.zip on your device that has Kodi (make sure to have the latest version of Kodi and to install and activate all of the plugins in Video InputStream on the official repository). Then install the repository via zip file, open our repository and install the add-on (it must be activated the auto-update on both repository and add-on, otherwise you don't get the newer updates)

We remind you that MarioFR Stream does not host, store, or distribute any media content on our servers. We operate solely as a search and indexing platform that aggregates publicly available information from various third-party services.

We function as a web scraping tool that helps users discover content across different platforms. The app indexes and organizes information that is already publicly accessible on the internet.
