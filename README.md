# mariofr-repo
1. Installazione su Kodi
Ora che i file sono online, configura Kodi:

Scarica lo ZIP della Repo: Dal tuo browser, vai nella cartella zips/repository.mariofr/ su GitHub e scarica il file repository.mariofr.zip.

Su Kodi:

Vai in Add-on > Installa da file zip e seleziona il file scaricato.

Vai in Installa da repository > MarioFR Repo > Add-on Video > MarioFR Stream > Installa.

Abilita InputStream: Assicurati che l'add-on InputStream Adaptive sia installato e abilitato su Kodi (necessario per i contenuti DASH/MPD e ClearKey).
